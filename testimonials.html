PennyWise

AI Agents for Smarter Finance

Problem Statement

Students often overspend despite setting budgets, struggling with impulsive buys, peer pressure, and poor tracking. This leads to financial stress and mismanaged essentials.

Our Solution

  • Automated expense tracking with AI agents
  • Smart budget recommendations
  • Spending insights and alerts
  • Adaptive learning from user behavior
  • Conversational AI assistant
  • Goal-based planning
  • Engaging UI with gamification

Technical Stack

  • Frontend: React.js or Flutter
  • Backend: Node.js / Python (Flask or FastAPI)
  • Database: MongoDB / PostgreSQL
  • AI: scikit-learn, TensorFlow, OpenAI APIs
  • Security: Firebase Auth, OAuth2, encryption
  • Deployment: Vercel, Render, Heroku
